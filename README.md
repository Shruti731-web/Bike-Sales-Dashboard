# Bike-Sales-Dashboard
## project overview
An interactive Bike Sales Dashboard built in Microsoft Excel to analyze customer purchasing trends. The project uses structured working sheets for data cleaning and transformation, converting raw data into meaningful visual insights.
## Dataset used 
- <a href="https://github.com/Shruti731-web/Bike-Sales-Dashboard/blob/main/Excel%20Project%20Dataset.xlsx">Dataset</a>
## KPI Questions
What is the total number of bikes sold?
What is the overall purchase rate (buyers vs non-buyers)?
Which income group has the highest bike purchases?
What is the average income of customers who purchased bikes?
Which age group contributes the most to bike sales?
What is the purchase distribution by region?
Which gender has a higher purchase rate?
How does marital status affect bike purchasing behavior?
What is the average number of children per buyer?
Which occupation or customer segment shows the highest sales?
What is the trend of bike purchases across different age categories?
What percentage of customers are frequent vs occasional buyers?
